How it works

Home Management Base Unit

High performance, low cost wireless "Smart Home Management" gateway with security, environment monitoring/ control and energy saving best fit for cloud based application service.

Remote Control Key Chain

4 button remote control, including "Home", "Away", "Disarm" and "Panic" functional button, allows us to apply different security logic to different situation.

Mini Door Magnet

A mini wireless door magnetic detector can be used to secure doors, windows, drawer, closet, cupboard or even refrigerator.

PIR Motion Detector

It is designed for space intruding protection and it detects the movement of human body heat within its effective coverage.

Support All Kinds of Platform

You can monitor your home status 24/7 anywhere any time with PC, desktops, mobile devices(iPhone, Android, WinPhone) and tablets.

1, 2, 3 Easy Installation:

  1. 1.

    Plug in base unit with Ethernet cable and install sensors using the install button at the back of the base unit.

  2. 2.

    Register user name and password with the registration ID provided. ( Email confirmation is required! )

  3. 3.

    Trigger a sensor signal and see the data being displayed on the user interface.
